{% trans "Mitä ovat henkilötietopyynnöt?" %}

{% blocktrans trimmed %} Henkilötietolain mukaan sinulla on oikeus pyytää kopio sinua koskevista tiedoista, tarkastettavaksi tai omaan käyttöösi. Henkilötiedoiksi lasketaan kaikki tieto mikä on liitettävissä sinuun. Pyyntöjä voi lähettää paitsi valtiolle ja kunnille, myös yrityksille ja järjestöille. {% endblocktrans %}

{% trans "Kuinka henkilötietopyynnöt toimivat?" %}

{% blocktrans trimmed %} Pyyntöjä voi lähettää kirjallisesti, kirjeitse tai sähköpostin välityksellä, henkilötietojasi tallettaneille tahoille. Pyyntöjä tehdessä sinun täytyy todistaa henkilöllisyytesi. Se miten tunnistautuminen tapahtuu riippuu organisaatiosta. Usein sinun on lähetettävä henkilökohtaisia tietoja, kuten asiakaskorttisi numero, tunnistautumista varten. Jotkin organisaatiot käyttävät varmentamiseen tunnistautumispalveluita, kuten verkkopankkitunnuksia. Tietojasi tallettaneella taholla on 40 päivää aikaa vastata pyyntöösi. Kopio sinusta koskevasta tiedosta lähetetään sinulle joko postissa tai sähköpostissa. Pyyntöihin vastaamisesta ei ole sallittua pyytää rahallista korvausta. {% endblocktrans %}

{% trans "Mitä Getyourdata.org tekee?" %}

{% blocktrans trimmed %} Tämän verkkosivun tarkoitus on helpottaa tietopyyntöjen tekemistä. Sivulle on tallennettuna tarvittavat yhteystiedot tärkeimmille tietopyyntöjen kohteille, kuten kauppaketjuille ja pankeille. Getyourdata.org auttaa sinua valmistamaan oikein muodoillun pyyntötekstin, ja voi lähettää sen sähköpostitse puolestasi tai tuottaa tulostettavan paperin jonka voit postittaa. Sivusto myös kerää muiden käyttäjien kokemuksia tietopyyntöjen tekemisestä yksittäisille organisaatioille, jotta pyynnöt onnistuisivat useammin. Sivusto on osa datam.me ryhmää, joka kehittää yhteisöä ja työkaluja henkilökohtaisen datan hallintaan. {% endblocktrans %}

{% trans "Mitä tietoja Getyourdata.org tallentaa minusta?" %}

{% blocktrans trimmed %} Sivusto ei vaadi rekisteröitymistä ja ei talleta henkilökohtaisia tietoja käyttäjistään. Vaikka tietopyyntöjen tekeminen jossain tapauksissa vaatii henkilökohtaisten yksityiskohtien kuten sosiaaliturvatunnusten täyttämistä, näitä tietoja ei talleteta palvelimellemme. Sivusto tallettaa web-palveluille tyypilliseen tapaan lokitiedostoa, josta näkyy kuinka mihin ajankohtaan käyttäjä on vieraillut verkkosivulla, ja missä osoitteissa hän on käynyt. {% endblocktrans %}
